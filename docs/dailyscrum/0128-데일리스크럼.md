## 3. 모임 등록 데이터 구조

필수 필드

```typescript{
price: 0, // 가격 (number, 필수) - 무료는 0
quantity: number, // 수량 (필수) - 모임 총 인원 수
// 제한 없는 경우: 999
// 예: 10명 모집 시 20~30명 받고 추리기
name: string, // 상품명 (필수) - 모임 제목
content: string, // 상품 설명 (필수) - 모임 설명
mainImages: string, // 모임 이미지
}
추가 필드 (extra 객체)
typescriptextra: {
region: string, // 지역
gender: string, // 성별
age: number, // 나이 (숫자 + "대")
survey: string, // 질문지
category: string, // 카테고리
}
```
