# 요구사항 정의서

## 1. 문서 정보

- 문서 목적: 취미공유 사이트 앱의 요구사항을 명확히 정의하여 이해관계자 간 공통 이해를 확보한다.
- 대상: 기획, 디자인, 프론트엔드
- 범위: 웹 기반 소셜 모임 플랫폼

## 2. 시스템 개요

사용자가 다양한 카테고리의 모임을 생성하고 참여할 수 있는 소셜 모임 플랫폼이다. 사용자는 지도 기반으로 주변 모임을 찾고, 관심 있는 모임에 신청하며, 주최자는 참여자를 관리할 수 있다. AI 기반 추천 시스템을 통해 개인화된 모임을 제안받을 수 있다.

## 3. 사용자 유형

- **비회원**: 모임 목록 및 상세 조회만 가능
- **회원**: 모임 생성, 신청, 북마크, 참여 등 모든 기능 이용 가능
- **주최자**: 모임 생성 및 참여자 관리 권한 보유

- 1순위 기능 개발 및 테스트

- 사용자 편의성 향상

- 3순위 기능 개발
- 고급 기능 추가

## 4. 제약 및 가정

- 브라우저 기반 웹 앱으로 제공한다.
- 초기 버전은 모바일 웹 우선으로 개발한다.
- 외부 지도 API(카카오맵 또는 구글맵)를 사용한다.
- 이미지 저장은 클라우드 스토리지를 활용한다.
